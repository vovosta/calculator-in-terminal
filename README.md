# Calculator in Terminal

A simple command-line calculator written in **C++**.

This project was created to practice the fundamentals of C++ programming, including functions, header files, source files, user input, and basic arithmetic operations.

## Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* Interactive terminal menu
* Simple and easy-to-read code

## Requirements

* Linux
* g++
* Git (optional)

## Build

Compile the project with:

```bash
g++ main.cpp calcul.cpp -o calculatrice
```

## Run

Execute the program:

```bash
./calculatrice
```

## Project Structure

```text
.
├── main.cpp      # Main program
├── calcul.cpp    # Calculator function implementations
├── calcul.h      # Function declarations
└── README.md
```
